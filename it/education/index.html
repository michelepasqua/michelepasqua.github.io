Tesi di Dottorato

Hyper Static Analysis of Programs  
An Abstract Interpretation-Based Framework for Hyperproperties Verification
Pasqua, M.

  PDF file         $\TeX$ ref

Durante il mio Dottorato ho sviluppato una metodologia per verificare, tramite analisi statica, hyperproprietà di programmi. In particolare, hyperproprietà correlate a flussi d'informazione, le quali sono cruciali in ambiti di sicurezza. L'approccio si basa su interpretazione astratta, un modello molto espressivo per approssimare la semantica di programmi. La mia tesi rappresenta il primo approccio sistematico che sfrutta l'interpretazione astratta per la verifica di hyperproprietà.
